# 6.2 Caracterização das Interfaces

São apresentados os wireframes das interfaces principais do SGCLC, acompanhados de uma descrição das suas funcionalidades e dos RFs que satisfazem.

## Ecrã de Login (RF01, RF04, RF05)

```
┌─────────────────────────────────────────────────────────────────┐
│                    🏪  SGCLC – QuickMart                        │
│              Sistema de Gestão de Cadeia de Lojas               │
│                                                                 │
│           ┌─────────────────────────────────────┐              │
│           │           INICIAR SESSÃO             │              │
│           │                                     │              │
│           │  Email: [________________________]  │              │
│           │  Password: [_____________________ ] │              │
│           │                                     │              │
│           │         [    ENTRAR    ]             │              │
│           │                                     │              │
│           └─────────────────────────────────────┘              │
└─────────────────────────────────────────────────────────────────┘
```
**Funcionalidades:** autenticação por email/password com trimming automático, mensagem de erro genérica, redirecciona para Dashboard do perfil correcto.

┌──────────────────────────────┬──────────────────────────────────┐
│  🛒 PONTO DE VENDA           │  CARRINHO  (3 itens)             │
│  Loja: Braga-Centro | Ana R. │                                  │
├──────────────────────────────┤  Produto       Qtd  P.   Total  │
│  [🔍 Filtrar...         ✕]  │  ─────────────────────────────  │

│  Produto  Stock  Preço  [+]  │  Red Bull       1  €1,80 €1,80  │
│  ─────────────────────────   │  Pão de Leite   2  €0,35 €0,70  │
│  🟡 Água 500ml   8  €0,60[+]│  ─────────────────────────────  │
│  ✅ Coca-Cola  145  €1,50[+]│  Subtotal:              €4,30   │
│  🟡 Red Bull     5  €1,80[+]│  Desconto (€):  [___]           │
│  ✅ Sumo Larj  230  €0,90[+]│  ─────────────────────────────  │
