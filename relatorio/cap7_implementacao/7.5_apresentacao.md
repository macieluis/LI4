# 7.5 Camada de Apresentação

## Blazor Server — Arquitectura de Componentes

A camada de apresentação é implementada com **ASP.NET Core Blazor Server**. Cada página da aplicação é um componente Razor (`.razor`) que combina HTML, CSS e C#. A renderização ocorre no servidor; o browser recebe apenas o diff do DOM via SignalR, o que garante tempos de resposta baixos mesmo em ligações lentas.

## NavMenu Adaptativo (RBAC)

O `NavMenu.razor` utiliza o `SessionService` para mostrar apenas os itens de menu permitidos ao papel actual:

```razor
@inject SessionService Session

@if (Session.HasRole("GestorCadeia") || Session.HasRole("GerenteLoja") || Session.HasRole("Funcionario"))
{
    <NavLink href="/dashboard">📊 Dashboard</NavLink>
}
@if (Session.HasRole("GerenteLoja") || Session.HasRole("Funcionario"))
{
    <NavLink href="/pos">🛒 Ponto de Venda</NavLink>
}
@if (Session.HasRole("GestorCadeia") || Session.HasRole("GerenteLoja"))
{
    <NavLink href="/stock">📦 Stock</NavLink>
    <NavLink href="/encomendas">📋 Encomendas</NavLink>
    <NavLink href="/faturas">🧾 Faturas</NavLink>
    <NavLink href="/relatorios">📈 Relatórios</NavLink>
}
@if (Session.HasRole("GestorCadeia"))
{
    <NavLink href="/consolidacao">🔄 Consolidação</NavLink>
    <NavLink href="/admin/utilizadores">⚙️ Utilizadores</NavLink>
}
```

## Componente POS — Filtro Client-Side

O POS (`POS/Index.razor`) implementa o filtro instantâneo client-side com a mesma função `Normalizar` utilizada nos repositórios:

```csharp
@code {
    private List<StockDto> todosProdutos = new();
    private List<StockDto> produtosFiltrados = new();
    private string termoPesquisa = "";

    private void AplicarFiltro(string termo)
    {
        termoPesquisa = termo;
        var n = Normalizar(termo);

        produtosFiltrados = string.IsNullOrWhiteSpace(n)
            ? todosProdutos
            : todosProdutos.Where(p =>
                Normalizar(p.NomeProduto).Contains(n) ||
                Normalizar(p.CodigoProduto).Contains(n) ||
                Normalizar(p.Categoria).Contains(n)
              ).ToList();
    }

    private static string Normalizar(string? s)
    {
        if (string.IsNullOrEmpty(s)) return "";
        var d = s.Normalize(NormalizationForm.FormD);
        var sb = new System.Text.StringBuilder();
        foreach (var c in d)
            if (CharUnicodeInfo.GetUnicodeCategory(c) != UnicodeCategory.NonSpacingMark)
                sb.Append(c);
        return sb.ToString().ToLower();
    }
}
```

## Padrão de Protecção de Rotas

Todas as páginas protegidas verificam a autenticação no evento `OnInitializedAsync()` e redirecionam para o login se a sessão não estiver activa:

```csharp
protected override async Task OnInitializedAsync()
{
    if (!Session.IsAuthenticated())
    {
        Navigation.NavigateTo("/login");
        return;
    }
    // Carregar dados da página...
}
```
